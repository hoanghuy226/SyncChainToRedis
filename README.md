# Khoá 1 DApp Bài 1

## Cài đặt

1. Cài NodeJS
2. Clone repo về
3. npm install

## Chạy thử

1. node index.js
2. Kiểm tra thông tin: http://localhost:4000
3. Add transaction: http://localhost:4000/addTx?from=0x01&to=0x02&amount=1.5&gas=0.1
